== Description ==
Engange with your visitors using whatsapp chat. A new feature allow your visitors to send a message on your whatsapp. Do you want to give a voice based user interface for your customer or users. then just configure the plugin and your users can search through your website using this plugin. Improve your speed of typing posts and pages by using your voice or speech. The plugin is usefull for you to automatically type into your WordPress using your voice.

== Installation ==
Follow the following steps to install a plugin or write to us at <a href="mailto:contact@wpandro.com">contact@wpandro.com</a> We will set it up for you.
  
1. Upload WpAssistance to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. New admin bar menu named as mic icon will be visible in your WordPress.
4. Click on the mic and start talking. 
5. The plugin will convert your speech into text which you can use to copy into your posts and pages.
6. For whatsapp chat you need to configure a valid whatsapp number in backend.
